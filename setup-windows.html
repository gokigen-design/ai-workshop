Windows ユーザーの方へ
Windows ではセキュリティソフトやシステム設定の影響で、インストール時にエラーが出ることがあります。事前にこのガイドの手順を済ませておいていただけると、当日スムーズに進められます。うまくいかない場合は、4/11 のキックオフで一緒に解決しますので、エラー画面のスクリーンショットを Slack でゆみまでお送りください。
1
サブスクリプション加入

まず Claude のアカウントを作って、Pro プラン（$20/月）に加入します。これだけで Claude Desktop アプリが使えるようになります。

1
下のリンクから claude.ai を開く
→ claude.ai を開く
2
「Sign up」でアカウント作成
Google ログインがおすすめです。
3
Settings → Billing へ
ログイン後、左下の自分の名前 → Settings → Billing へ進みます。
4
Pro プラン（$20/月）を選択して決済
クレジットカード情報を入力して完了。
プランについて
まずは $20/月 の Pro プランで十分です。使っていて「回数制限に引っかかるな」と感じたら、$100 や $200 のプランにいつでもアップグレードできます。キャンプ後にダウングレードも自由です。
2
Git for Windows をインストール

Claude Code はパソコン内部で「Git」というソフトを使います。これがないと Claude Code 機能が動きません。必ず先に入れてください。

1
Git のダウンロードページを開く
→ git-scm.com/downloads/win を開く
ページ上部の「Click here to download」をクリックしてダウンロードします。
2
ダウンロードしたファイルをダブルクリック
インストーラーが起動します。
3
すべて「Next」を押し続ける → Install → Finish
設定項目がたくさん出ますが、すべてデフォルトのまま「Next」で大丈夫です。
インストール確認方法
① Windows キーを押す
②「cmd」と入力して Enter
③ 黒い画面に git --version と打って Enter
④「git version 2.xx.x」と表示されれば成功です
3
Claude Desktop をインストール

ブラウザでも使えますが、Desktop アプリの方が快適です。ファイルのドラッグ＆ドロップや、コピー＆ペーストが楽になります。

1
Claude Desktop をダウンロード
→ claude.ai/download を開く
「Download for Windows」をクリック。Claude.Setup.exe がダウンロードされます。
2
ダウンロードしたファイルをダブルクリックして実行
インストールが自動的に始まります（数分かかることがあります）。
「WindowsによってPCが保護されました」と出たら？
→「詳細情報」をクリック →「実行」をクリックすれば進めます。
3
インストール完了後、自動で起動
インストールが終わると Claude が自動的に起動します。起動しない場合はスタートメニューから「Claude」を検索して開いてください。
4
ブラウザでログインする
アプリが起動すると、ブラウザが自動的に開いてログイン画面が表示されます。STEP 1 で作った Claude アカウントでログインしてください。
ブラウザが自動で開かない場合 → アプリ画面に表示される URL をコピーして、ブラウザに貼り付けてください。
5
「Code」タブを開いて、フォルダを選ぶ
アプリ上部の「Code」タブをクリック →「Select folder」で作業フォルダを選択。これで Claude Code が使える状態になります。
4
動作確認をしよう

インストールが完了したら、実際に Claude Desktop で簡単なメッセージを送って動作を確認してください。

1
まずは話しかけてみる
「こんにちは！ちゃんと動いていますか？」
→ Claude が返答してくれば OK です。
2
Claude Code 機能を試す
「Claude Code 機能を使って、デスクトップに test.txt というファイルを作ってください。中身は『テスト成功！』と書いてください。」
→ デスクトップに test.txt が作成されれば、Claude Code 機能も正常です。

うまくいかない時 / よくある質問
Q. 「WindowsによってPCが保護されました」と表示される
A. 「詳細情報」をクリック →「実行」をクリックしてください。これは Windows のセキュリティ機能で、新しいアプリに対して表示されるものです。
Q. git --version でエラーが出る / コマンドが認識されない
A. Git のインストール後、PCを再起動してからもう一度試してください。再起動しないとコマンドが認識されないことがあります。
Q. Desktop アプリが開かない / 動作しない
A. PC を再起動してからもう一度試してください。セキュリティソフト（ウイルスバスター、ノートンなど）がブロックしている可能性があります。セキュリティソフトの「例外設定」に Claude を追加してみてください。
Q. Google アカウントでログインできない
A. メールアドレスで新規アカウントを作成してください。
Q. 何をやってもダメ
A. 大丈夫です。エラー画面のスクリーンショットを Slack でゆみまでお送りください。
