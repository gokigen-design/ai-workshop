# 茶花AI勉強会 企画概要

## 基本情報
- **講座名（仮）**: ※タイトル検討中
- **主催**: あや + デザイナー仲間2名（計3名）
- **対象**: 茶花ホルダーさん（自分で講座やサービスを持っている一人起業家・フリーランス）
- **形式**: 6時間ワークショップ（10:00〜16:00 / ランチ休憩45分含む）/ リアル開催
- **制作物**: 全員共通で2つ程度の成果物を作る
- **料金**: 18,000円（モニター価格）/ 通常価格 25,000円
- **日程**: ※未定
- **場所**: ※未定（リアル会場）
- **定員**: ※未定

## コンセプト
「デザイナー3人が教える、AI臭くないものが作れるワークショップ」
パソコン苦手な女性起業家が、AIに話しかけるだけで自分のビジネスツールを作る体験。
デザイナー視点で「ダサくない」「ちゃんとおしゃれ」なものが作れるのが最大の差別化。

## 差別化ポイント
- **デザイナー3人体制** → AI臭くない、おしゃれなものが作れる
- **女性起業家向け** → 女性にウケるデザイン・テイストがわかるメンバー
- **非エンジニア** → 教える側もコード書けない。だから共感できる
- **実践型** → 座学じゃなく6時間ガッツリ手を動かす

## ワークショップの内容（6時間）
1. **Claude Codeのセットアップ**（使えるようにする）
2. **Claude Codeって何？できることの紹介**
3. **成果物①**: LINEプレゼント（診断ツール等）/ インスタ投稿の時短・モデリングリサーチツール
4. **成果物②**: LP制作 / スライド制作 / 自分専用アプリ
※ 全員共通で2つ程度作る予定

## 参加者が得られるもの
1. **時間** → 外注待ち・修正待ちがゼロ（自分で作れるから）
2. **お金** → LP外注15〜30万円 → 月額3,000円で自分で作れる
3. **スキル** → Claude Codeの基本操作（今後何でも作れる土台）
4. **完成物** → ワークショップ内で実際に動くものが2つ完成
5. **デザインのコツ** → AI臭くない仕上げ方（デザイナー視点）

## 開催の目的（運営側）
1. **認知** → ホルダーさんに私たちの存在を知ってもらう
2. **入口** → 「もっと難しいのやって」「時間ないからお願い」→ 制作依頼へ
3. **実績** → 教えている写真・〇名に教えた実績
4. **素材** → セミナー風景の写真・動画

## 訴求のキーワード
- 「デザイナー視点もりもり」
- 「あなたの家に1人デザイナー」
- 「AI臭さがなくなる」
- 「ダサいから抜けられる」
- 「忙しいから解放される」
- 「あれもこれもが形になる」
- 「外注費がなくなる」
- 「自分で全部できる」
- 「話しかけるだけ」
- 「非エンジニアでもできる」
- 「女性でもできる」

## あやのストーリー（共感ポイント）
- 服飾大学 → 飲食10年 → アパレル3年 → 出産 → Webデザイナー
- エクセル/ワード使えない、ブラインドタッチできない
- フォルダぐちゃぐちゃ
- でもClaude Codeで5つのツール/サイトを自力で制作
