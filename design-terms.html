デザインキーワード辞典

AIへの指示に使える日本語＋英語キーワード。英語キーワードをプロンプトに混ぜると精度が上がります。

1レイアウト・構図▾
ファーストビュー（FV）
First View / Above the Fold
ページを開いて最初に見える画面。ここで離脱されるかが決まる。
「FVにキャッチコピーとCTAボタンを入れて」
余白（ネガティブスペース）
Abundant Negative Space / White Space
要素まわりの空き。「何もない」が高級感を生む。
「もっと余白を取ってゆったりさせて」
Abundant negative space をプロンプトに入れるとAIが適切に余白を確保
非対称レイアウト
Asymmetrical Layout
左右のバランスをあえて崩す。動きと抜け感が出てプロっぽくなる。
「左右対称じゃなく、あえて非対称に」
Asymmetrical layout で完璧すぎるAIの対称性を崩せる
ブロークングリッド
Broken Grid
写真と文字を少し重ねるなど、枠にとらわれない配置。プロ感UP。
「写真とテキストを少し重ねるレイアウトにして」
Broken grid + Overlapping elements で雑誌風に
カラム
Column
縦方向の列。「2カラム＝横に2分割」のように使う。
「左に写真、右にテキストの2カラムで」
エディトリアルデザイン
Editorial Design
ファッション誌のような美しい配置。写真×文字の魅せ方が洗練。
「雑誌のような美しいレイアウトで」
Editorial design で一気に雑誌風の洗練感に
2配色・カラーパレット▾
トンマナ
Tone & Manner
デザイン全体の雰囲気・統一感。色・フォント・写真の世界観。
「全体のトンマナをナチュラルに揃えて」
くすみカラー
Muted Color Palette
彩度を落とした大人っぽい色合い。AIの「鮮やかすぎる色」を防ぐ。
「くすみカラーで大人っぽく」
Muted color palette はAIのビビッドすぎる配色を抑える必須ワード
アースカラー
Earth Tones
ベージュ・カーキ・テラコッタ。自然界にある温かい色。
「アースカラーで温かみのある配色に」
トーンオントーン
Tone on Tone
同じ色味で濃淡だけを変える配色。失敗なくまとまりが出る。
「ベージュのトーンオントーンでまとめて」
Tone on tone で色の統一感を簡単に指示できる
アクセントカラー
Accent Color
目を引くポイントに使う差し色。ボタンや強調部分に。
「アクセントカラーをコーラルピンクに」
温かいニュートラルカラー
Warm Neutral Colors
真っ白ではなく、アイボリーやグレージュ。肌馴染みの良い色。
「背景は真っ白じゃなくアイボリーで」
Warm neutral colors でAIの冷たい白を回避
3文字・タイポグラフィ▾
セリフ体 / 明朝体
Elegant Serif Font
文字に「ひげ」がある書体。高級感・信頼感・女性らしさ。
「見出しはセリフ体で上品に」
サンセリフ体 / ゴシック体
Clean Sans-serif Font
ひげがないシンプルな書体。モダン・親しみやすさ・実用性。
「本文はサンセリフでスッキリ見せて」
ゆったりとした字送り
Generous Letter Spacing
文字と文字の間隔を広げる。上品さ・優雅さ・余裕の演出。
「文字間を広めにして高級感を」
Generous letter spacing を入れるだけでプロっぽさが段違い
タイポグラフィ主役
Typography-driven
写真ではなく「美しい文字の配置」でデザインする手法。
「タイポグラフィ主役のデザインにして」
Typography-driven で文字で魅せるミニマルなデザインに
行間（ラインハイト）
Line Height
行と行の間隔。広いとゆったり読みやすい。
「行間をもう少し広げて読みやすくして」
4質感・あしらい（AI特有のテカテカ感を消す）▾
マットな質感
Matte Finish
テカリを消した上品な質感。画用紙やすりガラスのような仕上がり。
「マットな質感で上品に仕上げて」
Matte finish はAIのCG感・テカテカ感を消す最強ワード
柔らかいドロップシャドウ
Soft Drop Shadow
黒い濃い影ではなく「ふわっと広く薄い影」にして今っぽく。
「カードに柔らかい影をつけて」
Soft drop shadow でAIの強すぎる影を防止
水彩風のあしらい
Watercolor Accents
絵の具が滲んだような表現。背景や装飾に使うと柔らかい印象に。
「背景に水彩風のあしらいを入れて」
手描き風の要素
Hand-drawn Elements
矢印やアンダーラインに手書き感。親近感と人間味がプラス。
「ポイントに手描き風のアンダーラインを」
フィルムグレイン
Film Grain / Subtle Noise
フィルム写真のような微かなザラつき。デジタル臭さを消して人間味UP。
「写真にフィルムグレインをかけて」
Film grain + Subtle noise でAI画像の不自然さが消える
5動き・アニメーション（安っぽいパワポ感を避ける）▾
なめらかなフェードイン
Smooth Fade-in / Gentle Reveal
パッと出すのではなく、じわっと優しく要素を表示。上品な定番演出。
「セクションごとにフェードインで」
Smooth fade-in + Natural easing で人間味のある動きに
自然なイージング
Natural Easing
ロボットのような等速ではなく、フワッと始まりスッと止まる動き。
「動きは自然なイージングで」
Natural easing がないと機械的な動きになりがち
控えめなパララックス
Subtle Parallax Effect
背景と前景が違う速度で動き、奥行きを演出。「控えめ」がポイント。
「ヒーロー画像に控えめなパララックスを」
必ず Subtle をつけること。なしだとやりすぎになる
ゆったりした浮遊感
Gentle Floating Animation
写真やあしらいが呼吸するようにゆっくりフワフワと動く。
「装飾にゆったりした浮遊アニメをつけて」
ホバーエフェクト
Soft Hover Effect
マウスを乗せたとき、ふんわりと色や影が変わる心地よい反応。
「ボタンにソフトなホバーエフェクトを」
マイクロインタラクション
Micro-interactions
ボタンを押した時の小さな反応。細部の丁寧さがプロとの差。
「CTAにマイクロインタラクションを入れて」
6パーツ・要素▾
CTA（シーティーエー）
Call To Action
「申し込む」「購入」など行動を促すボタン。LPの最重要パーツ。
「CTAをもっと大きく目立たせて」
テスティモニアル
Testimonial
お客様の声セクション。顔写真＋コメントの構成が定番。
「お客様の声をテスティモニアルセクションで」
アコーディオン
Accordion
クリックで開閉する折りたたみUI。FAQ（よくある質問）に最適。
「FAQはアコーディオンにして」
フローティングボタン
Floating CTA
画面下に常に浮いている申込ボタン。スマホLPで必須。
「スマホは画面下にボタンを固定して」
カード
Card
情報をまとまりとして囲んだUI。Soft drop shadow で浮かせるのが今風。
「特徴を3つのカードに分けて並べて」
ソーシャルプルーフ
Social Proof
「〇〇名が参加」「メディア掲載」など信頼を高める社会的証明。
「参加者数やメディア実績を入れて信頼感を」
7デザイナーがよく使う「雰囲気の言葉」▾
世界観
Brand World
サービス全体から伝わる統一された雰囲気。色・写真・言葉が一体に。
「世界観が伝わるFVにして」
抜け感
Airy / Breathable
詰め込みすぎず、軽やかで風通しの良い印象。余白がカギ。
「もっと抜け感を出して」
こなれ感
Effortless
がんばりすぎず、でもおしゃれ。自然に洗練された印象。
「こなれ感のあるデザインにして」
メリハリ
Contrast / Rhythm
大小・太細の差をつけること。単調なデザインの解消に。
「見出しと本文でもっとメリハリを」
統一感
Consistency
色・フォント・余白のルールが揃っている。プロっぽさの源。
「全体の統一感を出して」
視線誘導
Eye Flow
読者の目線を意図通りに導くこと。CTAに自然と目が行くように。
「CTAに視線が行くように誘導して」

プロンプトの組み立て方

△ ざっくりすぎる 「おしゃれなLPを作って」
○ 日本語で具体的に 「ナチュラルな雰囲気のLPを作って。
　背景はアースカラー、見出しは明朝体で字間広め。
　余白たっぷりのゆったりした構成にして。
　CTAは丸ボタンでファーストビューに配置。
　各セクションはフェードインで表示して。」
○ 英語キーワードを混ぜると精度UP 「30代女性向けのヨガサロンのLPを作ります。
デザインのテイストは以下のキーワードを取り入れてください。」 Japandi style, Asymmetrical layout, Abundant negative space,
Earth tones, Matte finish, Elegant Serif font.
Smooth fade-in + Natural easing で上品で落ち着いた動きに。
○ 組み合わせで自分だけのスタイルに 「癒し・パステルベースだけど、
　見出しだけエレガント寄りのセリフ体にして、
　Muted color palette + Generous letter spacing で
　少し高級感も出したい。」
コツ
1. まず5つのスタイルから好みを1つ選ぶ
2. キーワード辞典から2〜3個追加する
3. 「もうちょっと〇〇にして」で微調整する

完璧な指示を最初から出す必要はありません。
スタイル名 + 2〜3個のキーワードだけで、AIのデザインは劇的に変わります。
