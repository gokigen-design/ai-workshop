# あや向け・受講生マスター作成手順（所要15分）

まず落ち着いて。順番通りにやれば誰でもできる内容です。
スプシを作るのは今日1回だけ。あとはずっとそれを使いまわします。

---

## このフォルダに入っているもの

| ファイル | 何これ |
|---|---|
| `受講生マスター_セットアップGAS.js` | スプシに色・罫線・ヘッダーを一発で入れる魔法のコード |
| `運用ルール.html` | チーム（ゆみ・ちゃかな）に渡す用の運用ガイド。ブラウザで開くだけ |
| `README_あやへの手順書.md` | これ。今読んでる |

---

## STEP 1｜Googleスプレッドシートを新規作成（2分）

1. Googleドライブを開く
2. 「新規」 > 「Googleスプレッドシート」 > 「空白のスプレッドシート」
3. 左上の「無題のスプレッドシート」をクリック → 名前を **「AIワークショップ 受講生マスター」** に変更
4. 置き場所は「AIワークショップ」フォルダの中がおすすめ

---

## STEP 2｜Apps Scriptを開く（1分）

1. スプレッドシートの上のメニューから **「拡張機能」 > 「Apps Script」** をクリック
2. 新しいタブが開く（少し待つ）
3. 最初から書いてある `function myFunction()...` は **全部消す**

---

## STEP 3｜コードを貼り付ける（1分）

1. このフォルダの `受講生マスター_セットアップGAS.js` をダブルクリックで開く
   - もしくは Finder から右クリック > テキストエディットで開く
2. **中身を全部選択してコピー**（Cmd + A → Cmd + C）
3. Apps Scriptの画面に戻って **貼り付け**（Cmd + V）
4. フロッピーアイコン（保存）をクリック、または Cmd + S

---

## STEP 4｜実行する（2分）

1. 画面上部にある関数選択のプルダウンで **`setupMasterSheet`** を選ぶ
2. その横の **「実行」** ボタンをクリック

### 初回だけ：権限の許可が出るので許可する

- 「承認が必要です」→ 「権限を確認」
- Googleアカウント選択 → 「詳細」 → 「（プロジェクト名）に移動（安全ではないページ）」
- 「許可」をクリック

※ これは「このGAS、スプシを編集していい？」という確認なので、自分のスプシに対して自分で許可するだけ。怪しいものではないので大丈夫

---

## STEP 5｜スプシに戻って完成を確認

1. 元のスプレッドシートのタブに戻る
2. 色分け＋罫線＋ヘッダー付きの受講生マスターが完成している！
3. 3行目の「やまださん」のサンプル行は、実際に1人目を入れるときに**上書きしてOK**

---

## これから毎回やること

### 受講生が増えたら
- 4行目、5行目……と下に追加していくだけ
- 色は最初から100行分入っているので、そのまま書けばOK

### 第2回（WS02）のワークショップが終わったら
- 同じスプシの続きに **WS02-01-xxx / WS02-02-xxx** と足していく
- 100行を超えそうになったら、GASエディタから **`addFiveRowsForNextRound`** を実行（5行追加される）

---

## チームメンバーへの共有

`運用ルール.html` をブラウザで開いてもらえば、ID体系もドライブのフォルダ構造も全部書いてある。
GitHub Pagesにアップすれば、LINEでURL1本送るだけで共有できます。

**アップする場合：**
```
ai-workshop リポジトリの
`student-ops/` フォルダに入れる → push → GitHub Pagesで自動公開
```

（これはあとでClaudeにやってもらえば数分で完了します）

---

## 困ったら

- GASの画面で赤いエラーが出る → エラー文のスクショを撮ってClaudeに見せる
- スプシが思った色にならない → もう一度 `setupMasterSheet` を実行すればリセットされる
- サンプル行を消したい → 3行目を選んで Delete

---

## ここまでできたら次にやること

1. **第1回のメンバー（いま手元にある感想・画像）をマスターに入れていく**
   - IDはあとで決めるので先に名前だけでもOK
2. **Googleドライブに `受講生素材/WS01_2026-04_第1回/` フォルダを作る**
3. **各受講生からもらった画像を `WS01-01-xxx/icon.png` の形で入れていく**
4. **スプシの画像URL欄に、ドライブの共有リンクを貼る**

全部終わったら、LPへの反映をClaudeに頼めば自動で差し替えます。
